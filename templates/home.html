About Project

This project is an EgSA summer 2020 training program project, tha main goal was to build a deep learning model that takes a satellite image and classify it to desert, plant or water.
To Know more please play the video

About model

model : This is a customized ResNet18 model.
Training : Trained on 375 images (with data augmentation) and 15% validation split with around 33.3% of total data on each class
Testing : Tested on 90 image
Accuracy : 98%
Validation loss : 0.6%
