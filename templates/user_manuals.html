Satellite images prediction model

Intended use

This model takes satellite images and classify it to one of these classes : [Desert, Plant, Water]

Requirements

No requirements needed more than an internet connection and the images you want to classify

  • examples of images:

Model Accessibility

Go to model page

User interface

  • browse button : to select images you want to upload from your device
  • submit button : to submit the selected images and upload them
  • display area : after you click the submit button a display area will apear with your images and the predicted class of each image

How to use

  • Click the browse button and select any number of images with any image formats.
  • Click the submit button.
  • Wait for the model to load and display your images along with the class of each one in the display area.
  • Repeat if you want to classify new images .

Tech

  • The model creation, training and testing was done on Kaggle
  • The website is hosted on heroku
  • The model weights file is hosted on Google Colab
  • The connection between heroku website and Google Colab classifing model is done using ngrok

Model specifications

If you want to know all the informations about the model you can go to Documentation page

  • Model : Cutomized ResNet18
  • Model accuracy : 98%
  • Model training : The model was trained on 357 images [around 33% in each class] with a 15% validation split and data. augmentation on training and validation data.
  • Model testing : The model was tested on 90 images [33.3% in each class].
